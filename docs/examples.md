# Example notebooks

End-to-end analyses on real datasets, rendered from the notebooks under
`examples/` in the repository. They are shown as-is (not re-executed during the
docs build); run them locally for an interactive version.

!!! note

    These notebooks come from the original MOSNA and are **untouched** by this
    fork — they demonstrate the same scientific pipeline, just running on the
    faster code path. To run them you will need the relevant optional backends
    (see [Getting started](getting_started.md)) and the corresponding datasets.

* [CODEX CTCL](notebooks/CODEX_CTCL.ipynb)
* [IMC Breast cancer](notebooks/IMC_Breast_cancer.ipynb)
* [Visium DLPFC](notebooks/Visium_DLPFC.ipynb)
